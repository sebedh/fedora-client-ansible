# Ansible to setup a Fedora workstation

## Requirements
- working incus
- ansible
- ansible community collection

# Instructions

## Step by step

1. Run `1.spawn_vm.sh` to setup a VM for development.
2. Run `2.ansible.sh` to actuall playbooks, you can any ansible flags to it.

## Open console

To open the console run `incus console <vm> --type vga`
